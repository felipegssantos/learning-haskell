# Learning Haskell

Here I document my efforts towards learning Haskell in specific, but also functional programming as a whole.

There are are two main ways I studied:

1. From the lecture notes of [UPenn's CIS194 course (Spring/2013)](https://www.seas.upenn.edu/~cis1940/spring13/lectures.html);
2. Doing exercises of the [typeclassopedia](https://wiki.haskell.org/index.php?title=Typeclassopedia), but also letting my own curiosity drive further explorations.
